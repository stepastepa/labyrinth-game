⚜️ Rules ⚜️

Первый игрок выбирается случайно. Затем очередь переходит по часовой стрелке по кругу.

Активный игрок — выделяется рамкой.

Сперва игрок должен поместить плитку на поле. Перед этим её можно поворачивать. Затем игрок должен кликнуть на любую доступную плитку, чтоб герой туда двигался. Далее очередь переходит к следующему игроку.

Цель игры - это собрать все артефакты, которые указаны в карточках игрока. Если игрок собрал все артефакты, то тогда активируется портал на стартовой плитке игрока. Для победы надо вернуться в свой портал.
